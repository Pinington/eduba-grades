# Eduba Grades

## Brief explanation

This app is meant to compute the minimum necessary passing grades for each subject. 
Learning to use Android Studio is the main goal of building this application.

## Prerequisites
To run this app you need:
 - Android 11 or above
 - An emulator with Android 11 or above

To build this app you need:
 - Android Studio Panda 4 | 2025.3.4 Patch 1
 - An emulator (built-in), or Android 11 and above phone
